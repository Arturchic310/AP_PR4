Корткий опис проєкту PR4_2:
------------------------------------------------------------------------------------------------------
Цей проєкт демонструє використання класу CompletableFuture у Java для виконання асинхронних обчислень.
При написанні коду було зроблено упір на використання CompletableFuture для виконання обчислень у
паралельних потоках, на реалізацію асинхронної роботи, яка забезпечувала б швидке виконання завдань та
мінімізацію блокувань основного потоку, на вивід, що форматує результати до двох знаків після коми.
------------------------------------------------------------------------------------------------------
Програма виконує такі основні кроки:
 - Генерація випадкових чисел: створення списку із 20 випадкових чисел у діапазоні від -100 до 100.
 - Розділення чисел: поділ списку на дві частини — числа з непарними індексами та числа з парними
індексами.
 - Обчислення мінімуму і максимуму: знаходження мінімального числа серед чисел із непарними індексами
та максимального серед чисел із парними індексами.
 - Обчислення результату: підсумовування мінімального значення непарних та максимального значення
парних чисел.
 - Вивід результату: відображення початкового списку чисел, а також результату обчислень.
