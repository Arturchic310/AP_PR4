Корткий опис проєкту PR4_1:
------------------------------------------------------------------------------------------------------
Цей проєкт демонструє використання класу CompletableFuture у Java для асинхронного програмування.
У проєкті показано роботу таких методів: supplyAsync(), thenAcceptAsync(), runAsync(), thenRunAsync()
------------------------------------------------------------------------------------------------------
Функціонал:
 - Асинхронна генерація масиву 3x3: Асинхронно генерується двовимірний масив 3x3 із випадковими
цілими числами.
 - Асинхронний вивід стовпців: Кожен стовпець згенерованого масиву виводиться асинхронно з
додатковим повідомленням.
 - Виконання фінальної задачі: Демонструється використання thenRunAsync() після виконання всіх
попередніх задач.
 - Виконання випадкової задачі: Демонструється використання RunAsync() для асинхронного виконання
задачі.
 - Вимірювання часу виконання: Для кожної асинхронної задачі логуються дані про час її виконання для
кращого розуміння тривалості.
